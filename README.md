# Documents

应用由三大模块组成:

- widget 数据展示形式
  - 响应式
- renderer 渲染器
- dataSource 数据源
- transform 数据转换

## widget

基础 widget:

- text
- image
- list
- tabs
- grid

rss: list + tabs
link: text / image

## dataSource

数据源: `用户输入` `Notion`
